기본적인 OS 구조
OS - 하드웨어와 소프트웨어 응용 프로그램을 연결하고 관리하는 **시스템 소프트 웨어**

커널(Kernal) - 운영체제의 핵심, 하드웨어와 프로그램 사이에 인터페이스 제공, 리소스 관리, 시스템 전반 통제

역할
- cpu, 메모리, 네트워크 등 하드웨어 리소스 / 운영체제, 프로그램, 드라이버 등 소프트웨어 리소스 관리
- 장치 드라이버 제공
- 사용자 인터페이스 제공
- 보안 
- 프로그램 실행
