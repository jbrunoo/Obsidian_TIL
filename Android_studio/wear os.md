화면 구성
Overlay - 모바일과 동일
Notification - 모바일과 동일
Compliation - 워치 페이스에서 바로 눈으로 정보를 확인 가능, 화면 누르면 해당 앱 환경으로 이동, 독립적인 액션도 수행 가능
Tile - 정보와 액션에 빠르게 엑세스 가능, 워치 페이스에서 좌우로 화면 쓸면 됨.
